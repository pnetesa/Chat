
--------------------
- Summary
--------------------

Chat application build on node.js and AngularJS.

Uses Redis store for saving data (Windows distrib can be downloaded here: http://tinyurl.com/l8sw3a4)

--------------------
- isDev property
--------------------

Property defined in backend's 'common.js' file and client's 'utils.js'. Enabling 'isDev' auto-fills credentials on login and registration page. On server side, attempt to create duplicate user or room will clean up all records when 'isDev' is set to true.
